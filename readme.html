djandchill.club about readme host
djandchill.club is a free-source developed to help DJ's use existing streaming and chat infrastructures to create a simple, anonymous, ad-free streaming experience for performers and their fans. we've built this tool to be as simple and straightforward as possible for hosts and viewers. read below for step by step guides: hosts:
  • create a twitch account
  • set up your streaming station (we recommend obs) - or stream from the twitch app on your phone (audio will be coming from the phone mic - not the best)
  • start your stream
  • go to djandchill.club/host
  • fill out the simple webform and submit
  • you will automatically be taken to a page with your twitch stream, and an anonymous chat powered by hack.chat
  • send fans the link to your room - djandchill.com/?your-room-title
viewers:
  • if you already have a link to a hosted room - just type the link into your browser, pick a nickname for the chat when prompted and press ok to be connected.
  • if you dont have a link - head over to djandchill.club to see a list of rooms currently active.
  • click on room and you'll be connected.
